# Screens Index

Every page in `src/pages/` has a dedicated mobile spec. All 29 screens listed
below are referenced from `ROADMAP.md` and each spec contains:

- Web → mobile **import mapping** (hooks kept as-is, UI swapped)
- Full list of **connected sub-components** with their file paths
- **Tailwind classNames preserved verbatim** from the web JSX
- Layout tree (top → bottom)
- Animation recipes (framer-motion → moti)
- Edge cases, native enhancements, performance, acceptance checklist
